app FocusNotch for MacOS
=======

macOS-приложение, превращающее челку экрана в интерактивную панель с таймером Pomodoro и музыкой.

## Блоки

### Блок вызова приложения
Появляется при наведении на область челки или на **фоновый блок** мини-таймера. Клик открывает основное окно приложения.
- Размер: 220×(`safeAreaInsets.top` + 8) pt
- В активном состоянии мини-таймера расширяется до 420pt (на всю ширину фонового блока)
- Скругление: только нижние углы, 16pt

### Фоновый блок (мини-таймер)
Отображается, когда запущен таймер Pomodoro. Показывает оставшееся время.
- Размер: 360×`safeAreaInsets.top` pt
- Позиция: `screenMidX - 180` (симметрично вокруг челки)
- Скругление: только нижние углы, 14pt (`BottomRoundedRect`)
- Рядом с челкой (справа) — иконка кофе во время перерыва

### Основное окно приложения (expanded)
Открывается кликом по **блоку вызова приложения**.
- Размер: 500×110 pt
- Скругление: только нижние углы, 16pt
- Содержит: таймер Pomodoro, элементы управления, интеграцию с музыкой

### Окно настроек
Отдельное окно с вкладками: General, Timer, Notification, About.
- Размер: 480×360 pt
- Тёмный фон (`NSColor(white: 0.12)`)

## Шрифт
Forza Thin — используется для времени в таймере, мини-таймере и всех надписей в настройках.

## Управление
- **Наведение на челку/фоновый блок** — появление блока вызова приложения
- **Клик** — открытие основного окна
- **Клик вне окна** — закрытие основного окна
